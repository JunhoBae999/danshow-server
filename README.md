# danshow-server
The server of danshow project. Implemented by Spring Framework including Spring Data Jpa, Spring Webflux(Webclient), Spring MVC, FFmpeg, Spring Security, etc.
Hosted by AWS EC2, RDS (MySql), S3 , etc
